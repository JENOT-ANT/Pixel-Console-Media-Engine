# Pixel-Console-Media-Engine
C++ library based on WinAPI useful to create games/visual applications in console window.

- engine.h  > library
- draw.cpp  > bonus example program (this program is to draw images in engine picture format)

## REQUIREMENTS:
- windows.h library
